Простой сайт для введения блога.
### Стек:
  - Django
  - HTML/CSS
  - Bootsrap 5
  - PostgeSQL (Взаимодействие через ORM)

### Функциональность
  - RSS
  - Поиск
  - Создание постов и категорий для них
  - Сортировка по категориям
  - Коментарии для поста
  - Пагинация
